YABTorrent by willemt

Yet another Bittorrent client/library written in C with a BSD license. The client has been intended to be used as a drop-in bittorrent library

Author

Willemt (himself@willemthiart.com)

Documentation

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/willemt/YABTorrent
